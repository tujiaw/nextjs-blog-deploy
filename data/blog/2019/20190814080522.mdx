# 安装
```
yum install iftop -y
```

# 查看网卡流量
```
iftop -i eth0
```

# 监控特定ip的流量
监控某个特定IP的带宽访问情况：
```
iftop -i eth0 -B -F 182.92.***.20
```
显示182.92.***.20这个IP与服务器的网卡eth1交互的数据量，单位是Byte。

# 退出
```
q
```

# 显示界面说明：
* ```"<="与"=>"```，表示的是流量的方向  
* "TX"：从网卡发出的流量  
* "RX"：网卡接收流量  
* "TOTAL"：网卡发送接收总流量  
* "cum"：iftop开始运行到当前时间点的总流量  
* "peak"：网卡流量峰值  
* "rates"：分别表示最近2s、10s、40s 的平均流量  

> eth0表示网卡，可能是eth0,eth1,eth2...
