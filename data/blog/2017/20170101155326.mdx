summary: 2017 年 1 月 1 日在 ubuntu 上安装最新版 nodejs，当前版本为 v6.9.2。通过 curl 下载、tar 解压、configure 配置、make 编译、make install 安装等步骤完成安装。 
---

如：当前版本是v6.9.2
> curl -0 https://nodejs.org/dist/v6.9.2/node-v6.9.2.tar.gz  
tar -xzvf node-v6.9.2.tar.gz  
cd node-v6.9.2  
./configure  
make  
make install
