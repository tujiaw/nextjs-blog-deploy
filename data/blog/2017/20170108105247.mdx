summary: 介绍了 mongoose 分页查询，PAGE_COUNT 为每页 item 数，page 是当前页面标号。通过 schema.find 结合 skip、limit 和 sort 实现分页查询，代码简洁高效。 
---

PAGE_COUNT：每个页面的item数  
page：当前页面标号

```
schema.find(query)  
.skip(PAGE_COUNT * (page - 1))  
.limit(PAGE_COUNT)  
.sort({ _id: -1 })  
.exec();
```
