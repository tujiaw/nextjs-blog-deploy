Qt窗口标题和exe设置图标，如工程名：joke。  

smile.ico是图标名
# 给窗口标题设置图标比较简单
```
QApplication a(argc, argv);
Dialog w;
w.setWindowTitle("Joke");
QIcon icon(":/rc/smile.ico");
w.setWindowIcon(icon);
w.show();
return a.exec();
```

# 给exe设置图标
* 在rc目录（与exe同一目录）下新建一个joke.rc；
* 用记事本打开粘贴如下文本
```
IDI_ICON1 ICON DISCARDABLE "smile.ico"
```
* 在pro文件中加入
```
RC_FILE = rc/joke.rc
```

如果是在vs中，就把rc文件加入到工程中，ico的路径是相对于rc文件的，可以在vs里编译下看看有没有问题


* 执行qmake，然后构建就可以了
