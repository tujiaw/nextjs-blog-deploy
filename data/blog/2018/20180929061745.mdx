请看一下代码：
```
double a = 5000.11;
int b = a * 100;
double c = a * 100;
```
b和c它们的值是什么？  
当你拿着b或者c去参与其他计算的时候，可能认为结果差不多。  

看如下结果：
```
b = 500010
c = 500010.99999999994
```
你会发现它们之间相差1，之所以会这样是因为double转换为int时会省略掉小数部分。我们可能还认为a乘以100应该为500011实际上结果是c。
