如果程序中使用的堆栈超过1M（默认）你就需要进行这个设置，但是设置太大又会造成虚拟内存过高。如果你明确的知道哪一个线程使用的
堆栈多高，可以只针对那个线程进行设置，这样不会影响到整个程序。

Qt线程设置堆栈大小函数是：
```
setStackSize(104857600); // 100M
```

递归调用次数过多也会造成堆栈大小过高。

visual studio 设置方法：
1. 右键工程属性
2. 链接器
3. 系统
4. 堆栈保留大小和堆栈提交大小，如：10485760（10M）
