关于我

我是一名AI研究员、工程师和教育者，热衷于构建能够理解和生成语言、图像和其他数据的系统。目前，我是Eureka Labs的创始人，这是一家AI+教育公司。

在我的职业生涯中，我一直处于AI研究和开发的前沿，从OpenAI的创始成员，到特斯拉的AI高级总监领导Autopilot计算机视觉团队，再到最近回到OpenAI，改进了ChatGPT上的GPT-4。

我坚信教育和知识共享的价值。我创建了关于AI和深度学习的流行教育内容，包括斯坦福大学的CS231n卷积神经网络课程，并运营一个YouTube频道，提供有关大型语言模型的技术讲座。

深度学习 计算机视觉 自然语言处理 大型语言模型 PyTorch TensorFlow 神经网络 Python Transformer架构 强化学习 AI研究 技术教育

主要项目

micrograd

一个微型标量自动微分引擎（具有强大功能！:)），它实现了在动态构建的DAG上的反向传播（反向模式自动微分）和一个具有类PyTorch API的小型神经网络库。

GitHub仓库

char-rnn

一个使用Torch中的LSTM/GRU/RNN构建的字符级语言模型。这个项目变得广受欢迎，与我的"循环神经网络的不合理有效性"博客文章相关。

GitHub仓库 博客文章

arxiv-sanity

一个驯服arXiv上压倒性论文洪流的工具。它允许研究人员发现相关论文，按相似度搜索/排序，并获得推荐。部署在arxiv-sanity.com上，后来重新实现为arxiv-sanity-lite。

GitHub仓库

neuraltalk2

一个早期的(lua)Torch图像标注项目，为图像生成自然语言描述。后来与Justin Johnson一起扩展为密集标注。

GitHub仓库

ConvNetJS

一个完全用JavaScript编写的深度学习库。实现了基于Web的演示，可以完全在浏览器中训练卷积神经网络。这导致了后来的项目，如tSNEJS、REINFORCEjs、recurrentjs和JS中的GAN。
